要实施基于NetApp的BeeGFS解决方案、请确保您的环境满足本文档中概述的技术要求。



== 要求概述

本页面详细介绍在 NetApp 构建模块上实施 BeeGFS 的硬件、软件和固件要求。要求分为以下几个部分：

* <<硬件要求>> - 文件节点（硬件/软件/固件）、块节点（硬件/固件）、网络和布线
* <<软件部署要求>> - Ansible 部署的组件
* <<Ansible控制节点要求>> - 控制节点软件版本




== 硬件要求

在开始之前，请确保您的硬件符合 BeeGFS on NetApp 解决方案的单个构建块设计的以下规范。根据客户要求，特定部署的确切组件可能会有所不同。每个文件节点应满足或超过这些规范，以实现预期性能。



=== 文件节点要求

|===
| 数量 | 规格 


| 2.  a| 
*Lenovo ThinkSystem SR国际 电脑公司

* *处理器：* 2x AMD EPYC 9124 16C 3.0 GHz（配置为两个 NUMA 区域）

* *管理：* 1GbE 端口，用于带内 OS 管理
* *BMC：* 带有 Redfish API 的 1GbE BMC，用于带外服务器管理
* *电源：* 双热插拔电源和性能风扇

* *操作系统：* Red Hat 9.4 Server Physical with High Availability (2 socket)
* *Linux 内核：* 5.14.0-427.42.1.el9_4.x86_64
* *HCA 固件：* FW：28.46.1006，PXE：3.8.0100，UEFI：14.39.0013


|===
